# 🚗 Sistema de Gestão de Veículos

> Um projeto prático de consola (terminal) focado em lógica de programação e gestão de dados. 💻

---

## ✨ Funcionalidades
* **➕ Cadastro de veículos:** Adicione facilmente novos veículos à sua frota.
* **📋 Listagem de veículos:** Visualize todos os veículos registados com clareza.
* **🗑️ Exclusão de veículos:** Remova registos desnecessários de forma rápida.
* **🖥️ Menu interativo:** Interface intuitiva e simples diretamente no terminal.

---

## 🛠 Tecnologias utilizadas
* **Linguagem:** `Java` ☕

---

## 🎯 Objetivo
Este projeto foi desenvolvido com o propósito de praticar:
* **Lógica de programação** e Orientação a Objetos.
* Manipulação de **Coleções (List, ArrayList)**.
* Estruturação de **Classes e Métodos**.
* Implementação de um **CRUD** (Create, Read, Update, Delete) básico.
* Boas práticas de organização de código.
